# Services
- [API Service](api-service.md) - A general-purpose HTTP client service built on the browser's `fetch` API.
- [IndexedDB Service](indexed-db-service.md) - A wrapper service for the IndexedDB API providing a Promise-based interface.
- [Local Storage Service](local-storage-service.md) - A utility service for interacting with the browser's `localStorage` API with JSON support.
- [Session Storage Service](session-storage-service.md) - A utility service for interacting with the browser's `sessionStorage` API with JSON support.
